Developing OpenWhisk Functions with Kotlin in IntelliJ.
I’ve implemented some code that shows how Apache OpenWhisk functions can be developed with Kotlin.
The Java code is built via Gradle and put into a Docker image which can be deployed to OpenWhisk cloud providers like the IBM Cloud.
